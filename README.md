### Built

**Cliniqly** &nbsp;·&nbsp; [cliniqly.tech](https://www.cliniqly.tech)<br/>
B2B SaaS for aesthetic clinic owners. A branded booking page per clinic, smart calendar, client records, and official WhatsApp reminders that cut no-shows. Built, shipped, and sold solo.

**Procurement Management System** &nbsp;·&nbsp; [eproc.psba.gop.pk](https://eproc.psba.gop.pk) &nbsp;·&nbsp; <sub>Punjab Sahulat Bazaars Authority</sub><br/>
Enterprise procurement platform digitizing the full tender lifecycle — creation, floating, evaluation, and bill execution — for MRS and non-MRS items used in constructing Sahulat Bazaars across every district in Punjab. Excel-based bulk import/export replacing manual data entry, and role-based access enforced across five government roles. Delivered within three months of joining. React, Node.js, Express, MongoDB.

**Alpha Grooming** &nbsp;·&nbsp; [alphagrooming.me](https://www.alphagrooming.me)<br/>
Full-stack e-commerce for men's grooming products, live in production. Admin panel, full product management, Stripe checkout, and on-page SEO down to the sitemap. React, Node.js, Express, MongoDB, Tailwind.

**Rent Around** &nbsp;·&nbsp; [live](https://rent-around-frontend.vercel.app)<br/>
Peer-to-peer rental platform — list an item, discover one, rent it from a neighbour. Sole developer of the Flutter app: every user flow, listing screen, and the admin dashboard. Final year project. Flutter, Node.js, Express, MongoDB.

**Blogify** &nbsp;·&nbsp; <sub>source available on request</sub><br/>
Blog platform built on Django REST Framework — CRUD, authentication, search, category filtering, a comment system, and a user dashboard. API-first, so any frontend can sit on top of it. Solo build.
